# 📝 React Simple To-Do List Project

A clean and minimal **React + Vite To-Do Application** with basic task management and **localStorage persistence**.  
Built for practice, understanding React fundamentals, and portfolio showcase.

---

## 🚀 Features

* ➕ Add new task
* 🗑️ Delete task
* 🚫 Prevent duplicate tasks
* 💾 Persistent storage using localStorage
* ⌨️ Add task using Enter key
* ⚡ Real-time UI updates without refresh

---

## 🛠️ Tech Stack

* React.js (Hooks: useState, useEffect)
* Vite (Fast development build tool)
* JavaScript (ES6+)
* CSS3 (Custom styling)
* LocalStorage API

## ⚙️ How to Run This Project

### 1. Clone the repository
```
git clone https://github.com/godhaniyaarbham2000-spec/ReactJS-Project-Simple-TO-DO-List.git
```

### 2. Move to project folder
```
cd ReactJS-Project-Simple-TO-DO-List
```

### 3. Install dependencies
```
npm install
```

### 4. Start development server
```
npm run dev
```
